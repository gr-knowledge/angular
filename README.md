# angular

### Instalação
```properties
npm install -g @angular/cli
```

### Comandos
| Comando | Descrição |
|---------|-----------------------------------|
|ng n\|new my-project | Novo projeto |
|ng n\|new my-project --routing --style scss | Novo projeto com rotas e scss |
|npm start | Emulador servidor web |


### Estrutura básica de arquivos
```
my-project
├── dist
├── src
|   ├── app
|   │   ├── core
|   │   ├── features
|   │   └── shared
|   ├── assets
|   │   └── (*.png, *.mp3)
|   ├── index.html
|   └── styles.css
└── package.json
```
