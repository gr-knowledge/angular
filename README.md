# angular

### Instalação
```properties
npm install -g @angular/cli
```

### Comandos
| Comando | Descrição |
|---------|-----------------------------------|
|ng n \| new my-project | Novo projeto |
|ng n \| new my-project --routing --style scss | Novo projeto com rotas e scss |
|npm start | Emulador servidor web |
